## Overview

The Account API allows you to manage your API account, check credit balance, monitor usage, and configure webhooks.

## Endpoints

| Endpoint | Description |
|----------|-------------|
| `GET /v1/account/info` | Get account information |
| `GET /v1/account/credits` | Check credit balance |
| `GET /v1/account/usage` | View usage statistics |
| `GET /v1/account/history` | Get request history |
| `POST /v1/account/webhook` | Configure webhooks |

---

## Get Account Info

Retrieve your account details and plan information.

```
GET /v1/account/info
```

<CodeGroup>
```bash cURL
curl -X GET "https://api.bouncewatch.com/v1/account/info" \
  -H "X-API-Key: YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://api.bouncewatch.com/v1/account/info', {
  headers: { 'X-API-Key': 'YOUR_API_KEY' }
});
const data = await response.json();
```

```python Python
response = requests.get(
    'https://api.bouncewatch.com/v1/account/info',
    headers={'X-API-Key': 'YOUR_API_KEY'}
)
```

```php PHP
$ch = curl_init('https://api.bouncewatch.com/v1/account/info');
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
curl_setopt($ch, CURLOPT_HTTPHEADER, ['X-API-Key: YOUR_API_KEY']);
$response = curl_exec($ch);
$data = json_decode($response, true);
```
</CodeGroup>

### Response

## Get Usage Statistics

```
GET /v1/account/usage
```

<ParamField query="period" type="string" default="month">
  Time period: `day`, `week`, or `month`
</ParamField>

<CodeGroup>
```bash cURL
curl -X GET "https://api.bouncewatch.com/v1/account/usage?period=month" \
  -H "X-API-Key: YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch(
  'https://api.bouncewatch.com/v1/account/usage?period=month',
```

## Configure Webhook

Set up webhooks for account notifications.

```
POST /v1/account/webhook
```

<ParamField body="webhook_url" type="string" required>
  Your webhook endpoint URL (must be HTTPS)
</ParamField>

<ParamField body="events" type="array" required>
  Events to subscribe to
</ParamField>

<ParamField body="secret" type="string">
  Optional webhook secret for signature verification
</ParamField>

### Available Events

| Event | Description |
|-------|-------------|
| `credit_low` | Credits below 20% |
| `credit_depleted` | Credits reached 0 |
| `enrichment.completed` | Realtime enrichment finished |
| `enrichment.failed` | Realtime enrichment failed |

<CodeGroup>
```bash cURL
curl -X POST "https://api.bouncewatch.com/v1/account/webhook" \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "webhook_url": "https://your-domain.com/webhooks/bouncewatch",
    "events": ["credit_low", "enrichment.completed"]
  }'
```

```javascript JavaScript
const response = await fetch('https://api.bouncewatch.com/v1/account/webhook', {
  method: 'POST',
  headers: {
    'X-API-Key': 'YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    webhook_url: 'https://your-domain.com/webhooks/bouncewatch',
    events: ['credit_low', 'enrichment.completed']
  })
});
```

```python Python
response = requests.post(
    'https://api.bouncewatch.com/v1/account/webhook',
    headers={
        'X-API-Key': 'YOUR_API_KEY',
        'Content-Type': 'application/json'
    },
    json={
        'webhook_url': 'https://your-domain.com/webhooks/bouncewatch',
        'events': ['credit_low', 'enrichment.completed']
    }
)
```

```php PHP
$ch = curl_init('https://api.bouncewatch.com/v1/account/webhook');
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
curl_setopt($ch, CURLOPT_POST, true);
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    'X-API-Key: YOUR_API_KEY',
    'Content-Type: application/json'
]);
curl_setopt($ch, CURLOPT_POSTFIELDS, json_encode([
    'webhook_url' => 'https://your-domain.com/webhooks/bouncewatch',
    'events' => ['credit_low', 'enrichment.completed']
]));
$response = curl_exec($ch);
$data = json_decode($response, true);
```
</CodeGroup>

### Response

```json
{
  "success": true,
  "data": {
    "webhook_id": "wh_345678",
    "webhook_url": "https://your-domain.com/webhooks/bouncewatch",
    "events": ["credit_low", "enrichment.completed"],
    "secret": "whsec_abc123xyz",
    "status": "active",
    "created_at": "2025-01-29T15:00:00Z"
  }
}
```

<Warning>
  Save the `secret` securely - you'll need it to verify webhook signatures.
</Warning>
