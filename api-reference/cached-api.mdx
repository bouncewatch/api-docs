## Overview

The Cached API returns company data instantly from our database. This is the default mode and is ideal for most use cases.

<Info>
  **Response time:** Under 250ms for most requests
</Info>

## Endpoint

<ParamField path="domain" type="string" required>
  Company domain (e.g., `stripe.com`, `openai.com`)
</ParamField>

```
GET /v1/company/{domain}
```

## Parameters

<ParamField query="enrich" type="string">
  Comma-separated list of enrichment modules to include.
  
  Available modules: `business`, `technology`, `funding`, `team`, `signals`, `competitors`
</ParamField>

## Request Examples

<Tabs>
  <Tab title="Base Data Only">
    Get basic company information (10 credits):
    
    <CodeGroup>
    ```bash cURL
    curl -X GET "https://api.bouncewatch.com/v1/company/stripe.com" \
      -H "X-API-Key: YOUR_API_KEY"
    ```
    
    ```javascript JavaScript
    const response = await fetch(
      'https://api.bouncewatch.com/v1/company/stripe.com',
      { headers: { 'X-API-Key': 'YOUR_API_KEY' } }
    );
    const data = await response.json();
    ```
    
    ```python Python
    import requests
    
    response = requests.get(
        'https://api.bouncewatch.com/v1/company/stripe.com',
        headers={'X-API-Key': 'YOUR_API_KEY'}
    )
    data = response.json()
    ```

    ```php PHP
    $ch = curl_init('https://api.bouncewatch.com/v1/company/stripe.com');
    curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
    curl_setopt($ch, CURLOPT_HTTPHEADER, ['X-API-Key: YOUR_API_KEY']);
    $response = curl_exec($ch);
    $data = json_decode($response, true);
    ```
    </CodeGroup>
  </Tab>

  <Tab title="Full Profile">
    Get all available data (50 credits):
    
    <CodeGroup>
    ```bash cURL
    curl -X GET "https://api.bouncewatch.com/v1/company/stripe.com?enrich=business,technology,funding,team,signals,competitors" \
      -H "X-API-Key: YOUR_API_KEY"
    ```
    
    ```javascript JavaScript
    const modules = ['business', 'technology', 'funding', 'team', 'signals', 'competitors'];
    const response = await fetch(
      `https://api.bouncewatch.com/v1/company/stripe.com?enrich=${modules.join(',')}`,
      { headers: { 'X-API-Key': 'YOUR_API_KEY' } }
    );
    ```
    
    ```python Python
    response = requests.get(
        'https://api.bouncewatch.com/v1/company/stripe.com',
        headers={'X-API-Key': 'YOUR_API_KEY'},
        params={'enrich': 'business,technology,funding,team,signals,competitors'}
    )
    ```

    ```php PHP
    $modules = ['business', 'technology', 'funding', 'team', 'signals', 'competitors'];
    $url = 'https://api.bouncewatch.com/v1/company/stripe.com?enrich=' . implode(',', $modules);
    $ch = curl_init($url);
    curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
    curl_setopt($ch, CURLOPT_HTTPHEADER, ['X-API-Key: YOUR_API_KEY']);
    $response = curl_exec($ch);
    $data = json_decode($response, true);
    ```
    </CodeGroup>
  </Tab>

## Response Fields

<AccordionGroup>
  <Accordion title="Base Company Data" icon="building" defaultOpen>
    | Field | Type | Description |
    |-------|------|-------------|
    | `name` | string | Company name |
    | `domain` | string | Primary domain |
    | `description` | string | Company description |
    | `tagline` | string | Company tagline |
    | `founded_year` | integer | Year founded |
    | `employee_count` | integer | Estimated employee count |
    | `employee_range` | string | Employee range bucket |
    | `headquarters` | object | HQ location details |
    | `social` | object | Social media links |
    | `contact` | object | Contact information |
  </Accordion>
  
  <Accordion title="Funding Data" icon="money-bill-trend-up">
    | Field | Type | Description |
    |-------|------|-------------|
    | `total_funding` | integer | Total funding in USD |
    | `last_funding_date` | string | Most recent funding date |
    | `last_funding_type` | string | Type of last round |
    | `valuation` | integer | Latest valuation in USD |
    | `rounds` | array | List of funding rounds |
    | `investors` | array | List of investors |
  </Accordion>
  
  <Accordion title="Team Data" icon="users">
    | Field | Type | Description |
    |-------|------|-------------|
    | `total_employees` | integer | Total employee count |
    | `hiring_status` | string | Current hiring status |
    | `open_positions` | integer | Number of open jobs |
    | `leadership` | array | Key team members |
    | `departments` | object | Employee breakdown by department |
  </Accordion>
  
  <Accordion title="Business Data" icon="chart-line">
    | Field | Type | Description |
    |-------|------|-------------|
    | `industry` | string | Primary industry |
    | `sub_industry` | string | Sub-industry |
    | `business_model` | string | B2B, B2C, etc. |
    | `target_market` | array | Target market segments |
    | `services` | array | Products/services offered |
  </Accordion>
  
  <Accordion title="Technology Data" icon="microchip">
    | Field | Type | Description |
    |-------|------|-------------|

  <Accordion title="Signals Data" icon="signal">
