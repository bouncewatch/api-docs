## What is BounceWatch API?

## Use Cases

<Tabs>
  <Tab title="Investors">
    **Due Diligence & Deal Flow**
    
    - Analyze startup funding history
    - Review team structure and leadership
    - Track growth signals
    
    ```bash
    GET /v1/company/startup.com?enrich=funding,team,signals
    ```
  </Tab>
  <Tab title="Sales Teams">
    **Lead Qualification & Prospecting**
    
    - Discover target company tech stacks
    - Identify growth stage
    - Find decision makers
    
    ```bash
    GET /v1/company/prospect.com?enrich=business,technology,team
    ```
  </Tab>
  <Tab title="Recruitment">
    **Talent Intelligence**
    
    - Track company hiring status
    - View open positions
    - Monitor team size changes
    
    ```bash
    GET /v1/company/target.com?enrich=team
    ```
  </Tab>
  <Tab title="Market Research">
    **Competitive Intelligence**
    
    - Perform competitor analysis
    - Track industry trends
    - Monitor technology adoption
    
    ```bash
    GET /v1/company/competitor.com?enrich=business,technology,competitors
    ```
  </Tab>
</Tabs>

## Quick Start

<Steps>
  <Step title="Get Your API Key">
    Create your API key from the [Dashboard](https://bouncewatch.com/api-panel/api-keys).
  </Step>
  <Step title="Make Your First Request">
    ```bash
    curl -X GET "https://api.bouncewatch.com/v1/company/stripe.com" \
      -H "X-API-Key: YOUR_API_KEY"
    ```
  </Step>
  <Step title="Add Enrichments">
    Add module parameters based on your needs.
    ```bash
    curl -X GET "https://api.bouncewatch.com/v1/company/stripe.com?enrich=funding,team" \
      -H "X-API-Key: YOUR_API_KEY"
    ```
  </Step>
</Steps>
